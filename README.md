- **ğŸ”‘ Sign up:** Allows new users to register for the system, creating a new account with email, user details & password.

- **ğŸ”‘ Sign in:** Secure authentication using Spring Security and JWT tokens, ensuring only authorized personnel can access the system.

- **ğŸ”‘ User Management:** Supports user registration, authentication, and authorization using Spring Security and JWT tokens, ensuring that only authorized personnel can access specific features.
